## How to create an API Key

1. Log in to the New Relic Console
2. Click your avatar

    ![Step 1](docs/assets/apikey_step_01.png){: style="max-width:100%"}

3. Go to `API Keys`

    ![Step 3](docs/assets/apikey_step_02.png){: style="max-width:100%"}

4. Click `Create a Key`

    ![Step 4](docs/assets/apikey_step_03.png){: style="max-width:100%"}

5. Select the account to use
6. Select `User` as Key type
7. Type a name for the Key
8. Type a notes (Optional)
9. Click `Create a key`

    ![Step 1](docs/assets/apikey_step_04.png){: style="max-width:100%"}

10. Click `Copy Key` and save it

    ![Step 1](docs/assets/apikey_step_05.png){: style="max-width:100%"}
